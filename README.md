# 🏥 Hospital Performance Analysis Dashboard

## Project Overview
This project analyzes hospital data using **SQL, Python, and Power BI** to gain insights into patient admissions, billing, medical conditions, hospitals, and insurance providers. An interactive dashboard was developed to support data-driven decision-making.

## Tools Used
- SQL
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- Excel

## Key Features
- Data Cleaning and Preprocessing
- SQL Data Analysis
- Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboard
- KPI Metrics and Business Insights

## KPIs
- Total Patients
- Total Hospitals
- Total Billing Amount
- Average Age
- Average Length of Stay
- Total Insurance Providers

## Dashboard Insights
- Patient distribution by medical condition
- Hospital-wise billing analysis
- Insurance provider analysis
- Admission type distribution
- Gender and blood type analysis
